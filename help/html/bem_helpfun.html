Functions and classes of the MNPBEM Toolbox

The toolbox includes a number of classes and functions.

  • BEM

Base

  • @bembase - Abstract base class for MNPBEM classes.
  • bemsolver.m - Select appropriate BEM solver using options.
  • dipole.m - Initialize dipole excitation.
  • electronbeam.m - Initialize electron excitation for EELS simulation.
  • greenfunction.m - Initialize Green function.
  • planewave.m - Initialize plane wave excitation.
  • spectrum.m - Initialize spectrum.

BEM

  • @bemiter - Base class for iterative BEM solvers.
  • @bemret - BEM solver for full Maxwell equations.
  • @bemretiter - Iterative BEM solver for full Maxwell equations.
  • @bemretlayer - BEM solver for full Maxwell equations and layer structure.
  • @bemretlayeriter - Iterative BEM solver for full Maxwell equations and layer structure.
  • @bemretmirror - BEM solver for full Maxwell equations with mirror symmetry.
  • @bemstat - BEM solver for quasistatic approximation.
  • @bemstatiter - Iterative BEM solver for quasistatic approximation.
  • @bemstateig - BEM solver for quasistatic approximation and eigenmode expansion.
  • @bemstateigmirror BEM solver for quasistatic approximation using mirror symmetry.
  • @bemstatlayer - BEM solver for quasistatic approximation and layer structure.
  • @bemstatmirror - BEM solver for quasistatic approximation with mirror symmetry.
  • plasmonmode.m - Compute plasmon eigenmodes for discretized surface.

Greenfun

  • @compgreenret - Green function for composite points and particle.
  • @compgreenrelayer - Green function for layer structure.
  • @compgreenretmirror - Green function for composite particles with mirror symmetry.
  • @compgreenstat - Green function for cquasistatic approximation.
  • @compgreenstatlayer - Green function for layer structure in quasistatic limit.
  • @compgreenstatmirror - Quasistatic Green function for composite particles with mirror symmetry.
  • @compgreentablayer - Green function for layer structure and different media.
  • @greenret - Green functions for solution of full Maxwell equations.
  • @greenretlayer - Green function for reflected part of layer structure.
  • @greenstat - Green functions for quasistatic approximation.
  • @greentablayer - Green function for layer structure.
  • @aca.compgreenret - Green function for full Maxwell's equations and ACA..
  • @aca.compgreenretlayer - Green function for layer structure using full Maxwell's equations and ACA.
  • @aca.compgreenstat - Green function quasistatic approximation using ACA.
  • @clustertree - Build cluster tree through bisection.
  • @hmatrix - Hierarchical matrix.

Material

  • @epsconst - Dielectric constant.
  • @epsdrude - Drude dielectric function.
  • @epstable - Interpolate from tabulated values of dielectric function.
  • @epsfun - Dielectric function using user-supplied function.

Mie

  • @miegans - Mie-Gans theory for ellipsoidal particle and quasistatic approximation.
  • @mieret - Mie theory for spherical particle using the full Maxwell equations.
  • @miestat - Mie theory for spherical particle using the quasistatic approximation.
  • miesolver.m - Initialize solver for Mie theory.

Misc

  • @bemplot - Plotting value arrays and vector functions within MNPBEM.
  • arrowplot.m - Plot vectors at given positions using arrows.
  • bemoptions.m - Set standard options for MNPBEM simulation.
  • coneplot.m - Plot vectors at given positions using cones.
  • coneplot2.m - Plot vectors at given positions using cones.
  • inner.m - Inner product between a vector and a matrix or tensor.
  • outer.m - Outer product between vector and tensor.
  • particlecursor.m - Find surface elements of discretized particle surface.
  • patchcurvature.m - Principal curvature values of a triangulated mesh.
  • subarray.m - Pass arguments to subsref.
  • units.m - Conversion between eV and nm.
  • vecnorm.m - Norm of vector array.
  • vecnormalize.m - Normalize vector array.

Particles

  • @comparticle - Compound of particles in a dielectric environment.
  • @comparticlemirror - Compound of particles with mirror symmetry in a dielectric environment.
  • @compoint - Compound of points in a dielectric environment.
  • @compound - Compound of points or particles within a dielectric environment.
  • @compstruct - Structure for compound of points or particles.
  • @compstructmirror - Structure for compound of points or particles with mirror symmetry.
  • @layerstructure - Dielectric layer structures.
  • @particle - Faces and vertices of discretized particle.
  • @point - Collection of points.
  • @polygon - 2D polygons for use with Mesh2d.

Particle shapes

  • @edgeprofile - Profile of smoothed edge for use with TRIPOLYGON.
  • @polygon3 - 3D polygon for extrusion of particles.
  • tricube.m - Cube particle with rounded edges.
  • tripolygon.m - 3d particle from polygon.
  • trirod.m - Faces and vertices for rod-shaped particle.
  • trisphere.m - Load points on sphere from file and perform triangulation.
  • trispherescale.m - Deform surface of sphere.
  • trispheresegment.m - Discretized surface of sphere.
  • tritorus.m - Faces and vertices of triangulated torus.

Simulation

Misc

  • @eelsbase - Base class for electron energy loss spectroscopy (EELS) simulations.
  • @meshfield - Compute electromagnetic fields for a grid of positions.

Quasistatic

  • @dipolestat - Excitation of an oscillating dipole in quasistatic approximation.
  • @dipolestatlayer - Excitation of an oscillating dipole in quasistatic approximation.
  • @dipolestatmirror - Excitation of an oscillating dipole using mirror symmetry.
  • @eelsstat - Excitation of an electron beam with a high kinetic energy.
  • @planewavestat - Plane wave excitation within quasistatic approximation.
  • @planewavestatlayer - Plane wave excitation within quasistatic approximation for layer structure.
  • @planewavestatmirror - Plane wave excitation within quasistatic approximation for mirror symmetry.
  • @spectrumstat - Compute far fields and scattering cross sections in quasistatic limit.
  • @spectrumstatlayer - Compute far fields and scattering cross sections for layer structure.

Retarded

  • @dipoleret - Excitation of an oscillating dipole.
  • @dipoleretlayer - Excitation of an oscillating dipole.
  • @dipoleretmirror - Excitation of an oscillating dipole.
  • @eelsret - Excitation of an electron beam with a high kinetic energy.
  • @planewaveret - Plane wave excitation for solution of full Maxwell equations.
  • @planewaveretlayer - Plane wave excitation for layer structure.
  • @planewaveretmirror - Plane wave excitation using mirror symmetry.
  • @spectrumret - Compute far fields and scattering cross sections.
  • @spectrumretlayer - Compute far fields and scattering cross sections for layer structure.
